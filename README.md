# Advertising Sales Prediction Using Linear Regression

This project analyzes the relationship between advertising spending (TV, Radio, Newspaper) and product sales using linear regression.

## Steps performed
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Train-test split (70/30)
- Linear regression modeling
- Model evaluation using R², MAE, RMSE, and MAPE

## Key insight
TV and Radio advertising show strong relationships with Sales, while Newspaper advertising has minimal predictive impact in this dataset.

- Hagan_HW.ipynb — analysis notebook
