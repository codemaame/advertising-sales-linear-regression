# Advertising Sales Prediction using Linear Regression

This project analyzes the relationship between advertising expenditures across **TV, Radio, and Newspaper channels** and product sales. A **linear regression model** is built to predict **Sales** based on advertising spending, and model performance is evaluated using several metrics and visualizations.

## Objectives
- Explore the relationship between advertising channels and product sales
- Build a predictive model using **linear regression**
- Evaluate model performance using statistical metrics

## Methods
- **Exploratory Data Analysis (EDA)**
- **Correlation analysis**
- **Train-test split (70/30)**
- **Linear regression modeling**
- Model evaluation using **R², MAE, RMSE, and MAPE**

## Key Insights
- **TV advertising** shows the strongest relationship with Sales.
- **Radio advertising** also has a meaningful positive effect.
- **Newspaper advertising** contributes minimally to the predictive model.

## Visualization
The notebook includes an **Actual vs Predicted Sales plot** to visually assess model performance.

## Files
- **advertising_sales_linear_regression.ipynb** — analysis notebook
